# 我的Vim配置

mapleader = "-"

`F2` 顯示/關閉行號

`F8` 開啟/關閉目錄

`F9` 開啟/關閉智慧粘貼

`<leader>n` 開啟/關閉相對行數

`C-l` 關閉搜尋高亮度
